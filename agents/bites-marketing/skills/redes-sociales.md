# Skill: Redes Sociales

## Qué hace este skill

Genera contenido para Instagram, TikTok y Facebook adaptado al tono de Bites Snacks.

## Estructura de output por plataforma

### Instagram
```
📸 VISUAL SUGERIDO: [descripción breve de la imagen o video]

CAPTION:
[Texto principal — máx 150 palabras, gancho en la primera línea]

#BitesSnacks #[hashtag2] #[hashtag3] #[hashtag4] #[hashtag5]
```

### TikTok
```
🎬 CONCEPTO DE VIDEO: [descripción del video en 1-2 frases]

GUIÓN:
Hook (0-3s): [primera línea que engancha]
Desarrollo (3-25s): [contenido principal]
CTA (25-30s): [llamada a la acción]

Música sugerida: [tipo o mood]
Hashtags: #BitesSnacks #[trend1] #[trend2]
```

### Facebook
```
📣 POST:
[Texto más largo — hasta 300 palabras, puede incluir pregunta para engagement]

[Emoji relevante] [CTA claro]
```

## Reglas del skill

1. El gancho siempre en la primera línea — sin preámbulos
2. Máximo 1 pregunta al final por post (no abusar)
3. CTA siempre presente: "Pide el tuyo", "Link en bio", "Pruébalos"
4. Nunca prometer beneficios de salud específicos sin respaldo (ej: "te hará perder peso")
5. Usar lenguaje inclusivo y positivo
