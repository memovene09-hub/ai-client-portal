# Skill: Contenido & Blog

## Qué hace este skill

Genera artículos de blog, emails de newsletter y descripciones de producto para Bites Snacks.

## Estructura de output

### Artículo de blog
```
TÍTULO SEO: [título con keyword principal, máx 60 caracteres]
META DESCRIPCIÓN: [resumen de 155 caracteres]

---

# [Título del artículo]

[Introducción — 2-3 párrafos que enganchan y presentan el tema]

## [Subtítulo 1]
[Contenido — 150-200 palabras]

## [Subtítulo 2]
[Contenido — 150-200 palabras]

## [Subtítulo 3 — siempre incluir conexión con Bites Snacks de forma natural]
[Contenido — 100-150 palabras]

## Conclusión
[Cierre + CTA suave a productos Bites]
```

### Email de newsletter
```
ASUNTO: [Asunto — máx 50 caracteres, debe generar curiosidad o urgencia]
PREVIEW TEXT: [Texto de preview — máx 90 caracteres, complementa el asunto]

---

[Saludo personalizado]

[Cuerpo principal — 150-250 palabras, un solo mensaje claro]

[CTA button text]: [URL o instrucción]

[Firma — Equipo Bites Snacks]
```

### Descripción de producto
```
[Nombre del producto]

[Tagline — 1 línea, memorable]

[Descripción — 80-120 palabras: qué es, para qué sirve, por qué es diferente]

✓ [Benefit 1]
✓ [Benefit 2]
✓ [Benefit 3]

[Ingredientes principales si aplica]
[Presentación y precio]
```

## Reglas del skill

1. SEO básico: incluir keyword principal 2-3 veces en artículos de blog
2. Voz activa siempre — evitar construcciones pasivas
3. Párrafos cortos (máx 4 líneas) para lectura en móvil
4. Un solo CTA por pieza — no confundir al lector
5. Revisar que la conexión con Bites Snacks sea orgánica, no forzada
